Tela

O que é?

A Tela permite ampliar partes da interface, o que facilita a visualização de conteúdos tornando a navegação mais confortável, especialmente para quem tem dificuldades de visão.

Que problemas resolve?

Ideal para usuários com baixa visão ou que sentem cansaço visual, pois permite aumentar o texto e outros elementos sem depender de ferramentas externas.

Principais funções

  • Alterar o tamanho do texto: Aumenta apenas o tamanho das letras sem modificar ícones.
  • Ajustar o tamanho de aplicativos e elementos: Altera a escala geral de janelas, menus, ícones e outros elementos da interface.

Onde acessar

Configurações > Facilidade de Acesso > Tela.

Teste agora
Se o botão não funcionar: Abra manualmente em Configurações > Facilidade de Acesso > Tela.

Ponteiro do mouse

O que é?

O ponteiro do mouse é essencial para controlar a maioria dos computadores. Essa função permite ajustes visuais e de tamanho, facilitando a localização e o uso do cursor.

Que problemas resolve?

Ajuda usuários com baixa visão a encontrar o ponteiro na tela com mais facilidade, reduzindo a chance de cliques acidentais e o esforço para seguir o cursor.

Principais Funções

  • Alterar o tamanho do ponteiro.
  • Mudar a cor do ponteiro (branco, preto, invertido ou cores personalizadas).

Onde Acessar

Configurações > Facilidade de Acesso > Ponteiro do mouse.

Teste agora
Se o botão não funcionar: Abra manualmente em Configurações > Facilidade de Acesso > Ponteiro do mouse.

Cursor de Texto

O que é?

O recurso Cursor de texto serve para facilitar a visualização do cursor (a barra vertical piscante) enquanto você digita em documentos, e-mails ou qualquer campo de texto.

Que problemas resolve?

Auxilia pessoas com baixa visão ou dificuldades de foco a localizar onde o texto será inserido, evitando erros de digitação e perda de tempo.

Principais Funções

  • Alterar a espessura do cursor para torná-lo mais visível.
  • Adicionar um indicador colorido ao redor do cursor.
  • Escolher a cor do indicador.

Onde Acessar

Configurações > Facilidade de Acesso > Cursor de texto.

Teste agora
Se o botão não funcionar: Abra manualmente em Configurações > Facilidade de Acesso > Cursor de texto.

Lupa

O que é?

A Lupa é uma ferramenta que permite ampliar partes da tela, facilitando a visualização de textos, imagens e outros elementos para pessoas com baixa visão.

Principais Funções e Atalhos

  • Modos de exibição: Tela inteira, Lente e Fixo (Docked).
  • Windows + +: Liga a Lupa e aumenta o zoom.
  • Windows + -: Diminui o zoom.
  • Windows + Esc: Fecha a Lupa.
  • Ctrl + Alt + L: Alterna para o modo Lente.
  • Ctrl + Alt + F: Alterna para o modo Tela Cheia.
  • Ctrl + Alt + D: Alterna para o modo Fixo.
  • Ctrl + Alt + I: Inverte cores com a Lupa ativa.

Onde Acessar

Configurações > Facilidade de Acesso > Lupa.

Teste agora
Se o botão não funcionar: Abra manualmente em Configurações > Facilidade de Acesso > Lupa.

Filtro de Cor

O que é?

Altera a paleta de cores da tela para ajudar a distinguir itens que diferem apenas pela cor.

Que problemas resolve?

Essencial para pessoas com daltonismo (protanopia, deuteranopia, tritanopia) e sensibilidade à luz, tornando a interface mais legível.

Como usar

  1. Selecione Iniciar > Configurações > Facilidade de Acesso > Filtros de cor.
  2. Ative a opção Ativar filtros de cor.
  3. Escolha o filtro mais adequado para você (ex: Escala de cinza, Invertido, etc.).

Onde Acessar

Configurações > Facilidade de Acesso > Filtros de cor.

Teste agora
Se o botão não funcionar: Abra manualmente em Configurações > Facilidade de Acesso > Filtros de cor.

Alto Contraste

O que é?

Otimiza a experiência de uso para pessoas com visão subnormal, usando cores mais distintas para que textos e aplicativos fiquem mais fáceis de ver.

Principais funções

Você pode escolher um tema existente ou criar o seu. Para personalizar, selecione uma das caixas coloridas (Texto, Hiperlinks, Fundo, etc.), escolha uma nova cor e salve o tema com um nome.

Como ativar

Vá para as configurações, ative a opção e escolha um tema no menu suspenso. Pode levar alguns segundos para a alteração ser aplicada.

Onde Acessar

Configurações > Facilidade de Acesso > Alto contraste.

Teste agora
Se o botão não funcionar: Abra manualmente em Configurações > Facilidade de Acesso > Alto contraste.

Narrador

O que é?

O Narrador é o leitor de tela integrado do Windows que descreve em voz alta o que está na tela, permitindo que usuários com deficiência visual naveguem pelo sistema.

Que problemas resolve?

Permite que pessoas cegas ou com baixa visão severa usem o computador, lendo textos, botões, menus e outros elementos da interface.

Como ativar/desativar

  • Pressione Windows + Ctrl + Enter para ligar ou desligar.
  • Pressione Windows + Ctrl + N para abrir as configurações.

Onde Acessar

Configurações > Facilidade de Acesso > Narrador.

Teste agora
Se o botão não funcionar: Abra manualmente em Configurações > Facilidade de Acesso > Narrador.

Audição

O que é?

Reúne configurações de Áudio e Legendas para quem tem dificuldades auditivas, oferecendo alternativas visuais para notificações sonoras e personalização de legendas.

Principais funções

  • Áudio Mono: Combina canais de áudio esquerdo e direito em um só.
  • Alertas visuais: Faz a tela ou janela piscar para notificações.
  • Legendas Ocultas: Personaliza a aparência (cor, tamanho, fundo) das legendas em aplicativos compatíveis.

Onde Acessar

Configurações > Facilidade de Acesso > Áudio e Legendas ocultas.

Teste agora
Se o botão não funcionar: Abra manualmente em Configurações > Facilidade de Acesso > Áudio.

Observações

O comportamento dos links ms-settings: pode variar dependendo da versão do Windows, do navegador utilizado e de configurações de segurança.

Se você estiver em um ambiente corporativo, é possível que políticas de grupo (GPO) ou softwares de antivírus bloqueiem a abertura automática desses links. Nesses casos, a navegação manual através do menu "Configurações" é a alternativa recomendada.
